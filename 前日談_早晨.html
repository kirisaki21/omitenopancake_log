前日談 ✦ 早晨

GM :

在離王都莫爾卡尼耶不遠處的郊區，一棟顯眼華麗的宅邸穩穩坐落著。這樣的別墅會畫有棲木的記號，實在是很稀有的事；那個記號甚至不是畫在大門的門扉上，而是畫在距離門扉有段距離的花園圍牆邊上。

鎮上的人們謠傳著，這是幢『魔女的宅邸』。或許艾絲塔和卡洛琳都曾經聽說過——又或者，就是因此而選擇這宅邸作為棲木？其實這裡並不是什麼魔女之屋，而是聲名顯赫的羅森家的其中一幢別墅。

如今，你們來到這棟宅邸已過去幾天了，宅邸的主人卻遲遲沒有出現，只有一個叫做南西的女僕貼身服侍著你們的所有起居。她似乎知道關於瑪奇莎的事情，不過在你們向她提出設置工坊的需求時她還是嚇了一跳。

你們剛來到棲木家的時候是怎樣的呢？而今天又是另一個在棲木家醒來的日子，你們會做些什麼呢？

瑟蕾娜 :

會來到這個家，是因為福金好像喜歡這裡。

福金是瑟蕾娜的烏鴉夥伴，敏銳的觀察力幫了很多忙，只要可行，瑟蕾娜多半會接納牠的意見。於是她寫了紙條，讓福金去交給房子裡的人。

這段時間就要在這裡打擾了。

但即使是住在這麼豪華的房子裡，瑟蕾娜仍然每天都懷抱著警覺之心醒來。

醒來時，會謹慎地維持不動，先觀察環境，等確定安全後才會起身，做每日必做的伸展運動。

卡洛琳．里希特霍芬 :

最初只有自己抵達時明明還是沒什麼人煙的大宅邸，雖然大得不可思議，但卡洛琳倒是待得自在，只因為這是羅森家的別墅，與過去里希特霍芬家有著同等級的氣派裝潢。
卡洛琳以來拜訪羅森的名義前來叨擾，雖然她不曉得南西為什麼沒多問什麼就輕易地放她進來，不過那不重要，只要能達成目的即可。

然而，靜謐的時光並不長，在不知不覺間這裡竟也變得熱鬧了起來，卡洛琳雖然覺得有些吵，但經過這幾天相處下來，大概也……快習慣了吧。

……如果某位不要一大早就大吵大鬧的話。卡洛琳剛享用著溫熱的紅茶，在心裡這麼想著。

伊芙琳 :

那是微雨的黃昏，伊芙琳本該前往王都，卻因為泥濘的水窪而耽誤了行程，她想起了早晨出行時塔羅牌所訴的預示——

一棟大的出奇的豪宅竟然是棲木？那或許也是個機會來讓她打理打理染上泥水的衣物。

然後，一待就是好幾天。畢竟床鋪實在太過舒服。伊芙琳一邊在內心想著沒辦法，一邊在床鋪上翻了個身。

艾絲塔‧羅莿 :

站在連身鏡前，艾絲塔認真進行每天的例行公事。

身上是請女僕準備好的、挑不出錯的精巧洋裝。
攏了攏前晚先抹上香膏，有淡淡香氣的暖澄捲髮。

擠眉弄眼地做出幾個看上去最能擄獲眾人芳心的可愛微笑，確認今天的自己也是最受人喜愛的艾絲塔推開房門。

「卡洛琳－－！」

急促但聽上去還是有所收斂、盡所能保持禮儀的腳步聲由遠到近，少女身影竄進飯廳。

卡洛琳．里希特霍芬 :

「早安，艾絲塔。」相較對方，卡洛琳的聲音聽起來冷漠多了。

不過，這已經比最初好很多，不知道從什麼時候開始，卡洛琳突然不再以羅莿來稱呼對方。

瑟蕾娜 :

完成今天早晨該做的伸展運動後，瑟蕾娜呼了口氣，和還窩在被子裡的伊芙琳搭話。

「伊芙琳？我們去吃早餐吧。」

伊芙琳 :

「好～」伊芙琳懶洋洋的欣賞完瑟蕾娜的早操，不情願的從床上慢慢地移動，溫暖的棉被彷彿還想留住她，或是她還蠻想被留住的。

「走吧。」打了個呵欠，她說著。

GM :

你們到了餐廳，餐桌上已經擺滿了適合早晨的餐點。這幾天一直在你們身邊的女僕南西佇立在一旁，注視著你們用餐。

瑟蕾娜 :

基本上不挑食，但吃的東西習慣選能吃飽的，拿了白麵包和奶油，然後抱著感恩的心多拿了火腿——老實說，肉可不是每天都能吃到的。

伊芙琳 :

伊芙琳還不太習慣被人盯著用餐，她偷偷瞄了南西幾眼，入座餐桌，剛起床的胃並不允許她大展身手，所以只是小口小口地、慢慢地、吃下鬆鬆軟軟的麵包。

卡洛琳．里希特霍芬 :

見其他人陸陸續續來到飯廳，卡洛琳只是維持著端正的坐姿點頭示意，當作打過招呼。

顯然她來得早，手中的麵包已享用掉大半，抹過果醬的餐刀就置放在一旁。

艾絲塔‧羅莿 :

「～♪」輕輕哼著曾經在劇場聽過的曲調，艾絲塔手裡的餐刀也沒停下動作。

雖說曾經家人會告誡在餐桌上必須保持優雅，甚至是不重要的閒聊都要避免，但都說是曾經了，現在又有誰會在意這麼多呢？

銀製刀具將肉排快速且俐落切成了好幾塊，完成後艾絲塔把部分肉塊分給了對面的瑟蕾娜和伊芙琳。

瑟蕾娜 :

「喔！謝、謝謝！」

伊芙琳 :

「謝謝。」伊芙琳開心的道謝，雖然因為剛起床再加上對方那面熟的臉讓她不太知道怎麼與艾絲塔相處。

不過，早餐吃肉排啊……

張開了嘴，她用叉子刺入多汁的肉排，吃了下去。

艾絲塔‧羅莿 :

對她來說面前的食物是再正常不過了，雖然能理解平民在看到佳餚的謹慎……不過猶豫即敗北！

「這就是貴族的……度量對吧？卡洛琳？」藉著抹果醬的動作，不知不覺間艾絲塔已經將沉重的餐椅挪到了距離卡洛琳半臂的位置，她悄聲和櫻髮少女說著讓人摸不著頭緒的悄悄話。

卡洛琳．里希特霍芬 :

「嗯？」比起那令人摸不著頭緒的問話，卡洛琳倒是暗自訝異艾絲塔什麼時候坐得近了。

「除了度量，我倒希望我們都保有貴族的禮儀。」卡洛琳把嘴裡的食物咀嚼下，這才暫時放下麵包，拿出一條潔淨的手帕，擦了擦艾絲塔的嘴角，「說話前應該注意嘴角的屑屑。」

瑟蕾娜 :

注意到卡洛琳的動作，連忙偷偷檢查自己的嘴角有沒有沾上食物碎屑。雖然不是貴族出身，但是與貴族同桌，果然還是多注意一點這種細節比較好。

也順便檢查一下伊芙琳的臉。

伊芙琳 :

伊芙琳注意到隔壁傳來的視線，有些尷尬地伸出舌頭舔去了肉排餘留下來的汁。

她的臉紅了點，不自在地伸手取了蘋果汁喝下。

瑟蕾娜 :

看到伊芙琳把自己打理乾淨了，放心地繼續吃東西，一面吃一面思考自己的箱子裡還有沒有缺什麼，得在出門前整理好才行。

艾絲塔‧羅莿 :

「嗯？」好幾秒才發現對方幫自己端正了禮儀，艾絲塔一把握住卡洛琳的手。

「太感謝啦－－就知道卡洛琳對我最好了！」

伊芙琳 :

伊芙琳不太自在的放下杯子，左看看卡洛琳，右看看艾絲塔。

這兩個人到底是什麼關係呢？

卡洛琳．里希特霍芬 :

卡洛琳默默地看著被艾絲塔攫住的右手，輕皺起眉頭，這下在對方鬆手以前，她尚未解決的半塊麵包只能靠左手來享用了。

「什！下、下次就沒有了。艾絲塔，你能專心吃早飯了嗎？」

艾絲塔‧羅莿 :

「嗯嗯嗯嗯，可以可以－－」鬆開握著對方的手，艾絲塔從盤內插起一塊肉排放到卡洛琳面前，接著用極快的速度把自己的肉排吃完，示意南西將瓷盤收走。

瑟蕾娜 :

「卡洛琳和艾絲塔，感情好像很好呢。」

已經吃飽了，在喝蘋果汁時看著兩人的互動感嘆。

艾絲塔‧羅莿 :

「我也這麼覺得！但是我跟瑟蕾娜的感情也很好呀！」

瑟蕾娜 :

「真的嗎！好開心。」

伊芙琳 :

「嗯……」她記得以前還在家裡時，沒有在鎮上看過卡洛琳，不過艾絲塔一直都是很招搖的人。

卡洛琳．里希特霍芬 :

「……」比起旁邊的兩人發表著喜悅的心情，卡洛琳無語地看著突然跑進自己盤裡的肉排。她可吃不下這麼多！艾絲塔這傢伙都不會先問她的感受！

雖然心中抱怨滿滿，卡洛琳最後還是切了一小塊肉，搭配著麵包將早飯享用完。

「南西小姐，我吃飽了。謝謝。」

伊芙琳 :

「……謝謝。」伊芙琳小聲地講了一句，擦了擦嘴表示自己也吃飽了。

　

你們結束了用餐，站在一旁的侍從們動了起來，開始收拾桌上的杯盤。

南西 :

南西在一旁靜待到你們用餐完畢，在一個合適的時機點，她向前一步，從圍裙口袋中拿出一張信紙。

「『大小姐們』。」她開口，這麼稱呼。你們早已被南西叮囑過，『羅森家遠道而來的親戚』這是你們在這個家的身分。

「主人捎來信件，不出意外，今天傍晚就會抵達本邸。今天晚上將會有一場迎接各位小姐的宴會，主人希望小姐們換上最好的禮服——一起享受這個特別的夜晚。」

伊芙琳 :

禮服？

在這裡待了一段時間，伊芙琳已經認定她不會有機會遇到羅森家的人，但宅子的主人非但要來，她還要參加宴會嗎？！

她沒有禮服……

瑟蕾娜 :

誠實地發出驚訝的呼聲，「咦，宴會嗎？但是禮服——」

已經很久沒有穿過華麗的衣服了！該去哪裡找呢？

卡洛琳．里希特霍芬 :

「有其他賓客嗎？」比起另外兩個人的擔心，卡洛琳似乎更在意與會貴賓。

艾絲塔‧羅莿 :

「嗯？」注意到兩人在聽到需著禮服時的模樣，艾絲塔眨了眨眼，對著伊芙琳和瑟蕾娜開口：「禮服我倒是帶了不少，如果不合身的話可以請人稍微調整一下晚上應該就能用了。」

南西 :

「請各位不必擔心冶裝，禮服的部分會由宅邸提供。只需要挑選符合您心意的，再稍作調整就行了。當然您也可以穿著自己帶來的。」

「說是宴會，但實際上只有大小姐們與主人，還有南西負責服侍而已。」南西謙卑地低下頭。

「主人想要與大小姐們度過一段私人的時光。」

伊芙琳 :

「可以的嗎？」這樣子的話真是幫大忙了。

瑟蕾娜 :

鬆了口氣，「那真是太好了，謝謝你們，艾絲塔，還有這裡的主人。」

伊芙琳 :

「這裡真的好厲害……」伊芙琳有些感嘆，能夠一直住在這種地方，一定很幸福。

艾絲塔‧羅莿 :

在聽見會提供禮服，艾絲塔發出驚喜的聲音。

「會有這陣子流行的透膚材質嗎？有沒有群擺是蕾絲材質的禮服？我出門的時候忘記帶了。」喃喃自語同時又想起了什麼。

「說起來，宅邸主人……是女性嗎？」這可取決於晚上禮服風格，要是搶了人風采就不好了。

瑟蕾娜 :

「說起來我有點好奇，」小心地發問，「這裡的主人願意讓我們借住在這裡，真的很感謝⋯⋯但像南西的主人這樣慷慨的人並不多，為什麼會願意讓我們借住呢？」

卡洛琳．里希特霍芬 :

「既然是羅森的願望，那好吧，我可以勉為其難出席一下。」賓客名單意外地少，卡洛琳思量了一下，這才答應宴會邀請。

「禮服的話就不用了，我自己有準備。」

至於羅森的事，她倒沒那麼好奇。畢竟現在再怎麼說，也不過是南西口中的介紹。反正今晚見到羅森，動用點真知魔法，她就能更直接地知道她想知道的事。

南西 :

「這棟宅邸的主人名叫『克拉麗莎．格雷姆．羅森』。」南西維持著低頭的姿勢繼續說明。

「主人理所當然的……是一名女性。今晚主人也會盛裝出席，不過還請各位小姐不必太拘束，主人的性格不會過於計較禮數。」

「瑟蕾娜大小姐，您這麼說就太見外了。身為羅森家的遠房親戚，南西相信不只是主人，只要冠名『羅森』的大人都會慷慨接待的。」說到這時，南西抬起頭來，向瑟蕾娜眨了眨眼，又看了一眼其他還在忙碌的侍從們。

瑟蕾娜 :

「那真是，承蒙照顧⋯⋯」有點受寵若驚，「晚上的宴會，如果我們的參加能讓羅森夫人高興的話就好了。」

伊芙琳 :

「……」微微點了個頭，她寄人籬下，會好好表現的。

南西 :

「……」聽見那個稱呼，南西偏頭，稍作思考。

「我想主人不會喜歡那個稱呼的，瑟蕾娜小姐……非常抱歉，如果可以的話請您稱呼主人『克拉麗莎小姐』就行了。」南西略帶歉意地笑了笑。

「曾經的主人或許會很高興被稱為誰的夫人，但是如今，能讓她冠上這個稱謂的人已經不在了。」

瑟蕾娜 :

「啊！原來是這樣⋯⋯是我冒犯了。」吃了一驚，連忙道歉，「我會記住稱呼，不會再叫錯的。」

艾絲塔‧羅莿 :

「嗯……這樣……」從南西話裡很快意識到了些什麼，但沒有多問，只是點點頭。

內心默念了下被更名成艾絲塔‧羅森的新名字，感覺好像也是挺順耳的？

「那接下來……卡洛琳‧羅森要一起去哪邊散步嗎？」說話同時順手各勾起了瑟蕾娜和伊芙琳的手，好像在吃早餐期間大家已經達成了什麼不存在的約定般。

瑟蕾娜 :

「要去散步嗎？」沒有聽說過耶！

卡洛琳．里希特霍芬 :

看來自己跟克拉麗莎有共同點，這點倒是讓卡洛琳挺滿意的。

雖然如此，當艾絲塔突然稱呼自己為羅森的時候她還是很不習慣。

「叫我卡洛琳就可以了。」比起答應散步這種事，她先是厲聲糾正了一番，就算她已不再以里希特霍芬自稱，不代表她能接受其他人的姓。

伊芙琳 :

「行吧……」被勾住的手讓伊芙琳整個人僵在原地，她還沒習慣那麼親暱的舉動！

艾絲塔‧羅莿 :

「欸！沒有要去散步嗎？我以為瑟蕾娜會一起去……」噘起嘴眨了眨眼，試圖擠出不存在的眼淚。

瑟蕾娜 :

更慌亂了，「不是、我的意思是⋯⋯嗯，去吧！」

艾絲塔‧羅莿 :

在處理（？）完瑟蕾娜，艾絲塔拉著兩人又是一個箭步湊近卡洛琳。

「嗯嗯，那我們等下是要去花園呢……還是湖邊……？」雖然沒說出口，但滿臉的愉快表達了：嘿嘿我知道卡洛琳也覺得加上姓氏的話太生疏了，果然卡洛琳也喜歡我！

除了自我感覺良好外，也完全忽略了對方根本沒回答去不去。

卡洛琳．里希特霍芬 :

「我想去鎮上。」艾絲塔的突然湊近，令卡洛琳不得細想，轉而下意識地把自己今天的安排說溜了嘴。
