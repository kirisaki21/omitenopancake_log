回憶的鬆餅

前日談 ✦ 王都的街上

GM :

你們決定要去鎮上。這裡土黃色街道的兩側都是磚造建築，門、窗、墻壁點綴著鮮艷的顏色，有著統一感的同時帶有活潑，讓人百看不厭。

王都大部分的房屋平而低矮，即使從港口的船隻眺望過來，主要建築物和王城也非常顯眼。

現在的時間是早晨，有許多商店都已經開始營業了。你們想要去哪裡或是做些甚麼呢？

卡洛琳．里希特霍芬 :

「到城鎮啦，那麼，晚上見。」卡洛琳也不知道他們和自己來鎮上幹麻，不過她打算去買今晚宴會的鞋子，於是探頭確認了下商店街區的方向。

瑟蕾娜 :

「我想去廣場看看。」從以前就非常習慣在廣場表演，打算從廣場開始，如果在那邊稍微表演一下聚集人潮，說不定還可以打聽到貝琳達的下落。

伊芙琳 :

聽見瑟蕾娜要去廣場，她便打定主意也跟著去廣場好了。如果是廣場的話，應該很熱鬧，也能體驗一下這裡的風情吧？

艾絲塔‧羅莿 :

「那我……」本來也想一起到廣場，但不遠處巷口那不起眼的招牌吸引了艾絲塔的注意力。

「我也去逛逛好了！」這麼說著，一溜煙地竄進了人潮中。

☜王都街道分流｜卡洛琳、艾絲塔

GM :

瑟蕾娜和伊芙琳兩人牽著手，不一會就在商店街區的一家時髦的店門口看見了熟悉的身影。

艾絲塔‧羅莿 :

「這可是獨家機密，錯過這個村就沒這個店啦。」艾絲塔抬手，裝模作樣地比出了噤聲的手勢。

「不過其實我還多買了一些，如果卡洛琳還想……啊、」走在卡洛琳身旁，話還沒說完就看到了不遠處的三人。

艾絲塔‧羅莿 :

「瑟蕾娜、伊芙琳、南西－－這邊這邊！」放聲喊著的同時朝街道另一端用力揮手、沒在管周圍人潮的艾絲塔吸引了不少目光。

伊芙琳 :

伊芙琳聽見艾絲塔的大喊抖了一下，無措地望著對方。

瑟蕾娜 :

看見艾絲塔的大動作，先是被聲音小小嚇了一跳，才拉著伊芙琳快步走過去。

「艾絲塔、卡洛琳，你們買到想要的東西了嗎？」

卡洛琳．里希特霍芬 :

嘴裡才剛放進一塊小餅乾，便直擊艾絲塔的吶喊，身旁的卡洛琳不悅地輕皺眉頭，卻苦於食不語的禮儀，無法出言指正。

南西 :

南西在後頭小幅度的，輕輕揮了揮手。

卡洛琳．里希特霍芬 :

將嘴裡的食物細嚼慢咽吞了下去，卡洛琳這才拿出手帕擦擦嘴，「……買到了。」

艾絲塔‧羅莿 :

「獲得了珍貴的東西！」胸前的墜飾在陽光折射下閃著光。

艾絲塔話說完，從口袋拿出三包袋裝糖果放到各自手裡，「這是伊芙琳、這是瑟蕾娜還有南西的！」

糖果是鎮內部分店家都有販售的經典產品，卻又和剛才交給卡洛琳的點心不太相同。

瑟蕾娜 :

用驚訝的表情收下糖果，「謝謝！」

伊芙琳 :

「謝謝。」伊芙琳的眼睜得大大的，收下了糖果。自由的手好奇地在包裝上捏了捏。

南西 :

「唔……」南西似乎本來想拒絕的，但在猶豫了一陣子之後，還是收下了。

「謝謝小姐。」她帶著微笑，將糖果收進了圍裙中。

南西 :

「那麼小姐們，是不是該回到宅邸進行晚宴的準備了？」南西很快地再次端正回僕人的身分，稍稍鞠躬向大家詢問。

伊芙琳 :

伊芙琳悄悄地望向大家。

瑟蕾娜 :

「好的。」乖巧回答。

艾絲塔‧羅莿 :

「如果大家都已經結束行程的話－－」

卡洛琳．里希特霍芬 :

雖然最初說要來鎮上的是她，但除了鞋子外，不知不覺身上也多了不少東西。

卡洛琳想了想，點點頭。她可不希望自己倉促出席舞會，盛裝打扮還是必要的。

至於她原本要辦的事，就改日再說吧。

南西 :

獲得各位的應允後，南西隨侍在你們身後，與大家一起回到了宅邸。
